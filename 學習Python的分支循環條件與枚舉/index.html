什麼是表達式

表達式(Expression)是運算符(operator)和j操作數(operand)析構成的序列

表達式的優先級

運算子 說明
(…)、[…]、{…}、… 管串、串列、辭典、轉換成字串
x[i]、x[i:j]、x.y、x(…)、x.y(…) 索引運算、切片運算、名稱限定用法、函式/方法呼叫
+x、-x、~x x不變、負號之x、x之補數
x * y、x / y、x % y、x // y 相乘、相除、餘數、相除取floor()
x + y、x – y 相加、相減
x << y、x >> y 位移運算
x & y AND位元運算
x ^ y XOR位元運算
x | y OR位元運算
<、>、<=、>=、==、<>、!=,、is、is not、in、not in 比較運算、本體測試、序列元素關係測試
not x 否定布林運算
x and y AND布林運算
x or y OR布林運算

not > and > or
