LINE Login 開發

因為要以line來登人我們的網站，所以要來申請

  • 設定
  1. 去 developers.line 註冊 login，記住 Channel ID,記住Channel secret
  2. 設定 Callback URL，這個和程式要一樣，如果是對外要https, 測試時可以使用localIP[ex. 192.168.2.144]
    注意在linelogin的Callback URL要一樣和上面登人的按鈕的網址要一樣
  • 程式
  1. 取得授權碼 code
  2. 以回傳的授權碼再向 Line 取用戶資料
