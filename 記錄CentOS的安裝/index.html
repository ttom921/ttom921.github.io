    安裝vsftp

    我們從 CentOS 官方套件庫直接安裝就可以了。 啟動服務
    不必修改任何設定值，就可以正常的啟動 vsFTPd 服務 預設的 FTP 根目錄在 /var/ftp。
    我們可以視需求，決定是否讓服務在開機時就自動啟動。 主設定檔及相關設定檔的預設儲存目錄在 /etc/vsftpd。 其他相關設定
    防火牆 如果有啟用被動模式的話，再加上指派給它的通訊埠範圍。 相關設定檔
    /etc/vsftpd/ftpusers
    這個檔案用來列出絕對禁止登入 FTP Server 的本機使用者，預設的名單有 root 跟一些常見的本機服務帳號，像是 mail、sync、shutdown … 之類的。
