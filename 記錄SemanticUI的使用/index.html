  4. 更新 @angular/cli

  5. 更新 Angular 相關套件版本
    • ng update @angular/core

      加入 Semantic UI

      安裝 typescript 安裝 Gulp
      有個坑，它的less的版有限定less 2.7.3
      先移除新的less 3.0.4
      $ npm remove less@3.0.4
      再安裝 less 2.7.3
      $ npm install less@2.7.3
