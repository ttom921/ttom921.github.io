JSON

什麼是JSON (JavaScript Object Notation)

  • 一種輕量級的數據交換格式
  • JSON是一種數據格式
  • 字符串是JSON的表現形式
  • 符合JSON格式的字符串叫做JSON字符串
    • {a:”qiyue”} 不是JSON字符串
    • {“name”:”qiyue”} 是JSON字符串

JSON的優勢

  • 易于閱讀
  • 易于解析
  • 網絡傳輸效率高
  • 跨語言交換數據
