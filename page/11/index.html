我想這遊戲應該有玩過，它的規則

  • 4碼不重覆數字
  • 有相同的數字為B,有相同的數字且位置一樣為A

這樣大家應該看不懂(我也看不懂)，看例子比較快
你設定的數字為5678
電腦猜1234 你要輸入0A0B
電腦猜4567 你要輸入0A3B
電腦猜5478 你要輸入3A0B
電腦猜5479 你要輸入2A0B
電腦猜5678 你要輸入4A0B
電腦會猜中你的數字

If you’re using markdown for building websites, you may specify a URL prefix for image preview in local computer with property typora-root-url in YAML Front Matters.
