配置identity server

在startup.cs 裏編輯ConfigureServices方法
這裏要把IdentityServer註冊到容器中，還要至少對其配置三點內容
1.1 哪些API可以使用這個authorization server
1.2 那些客戶端Client(應用)可以使用這個authorization server
1.3 指定可以使用authorization server授權的用戶
將上面做成一個配置文件
建立 Configuration/InMemoryConfiguration.cs

ApiResource:這裏指定了name和display name, 以後api使用authorization server時候，這個name一定要一樣
Client:屬性太多，先指定儿個
ClientSecrets:是用來取得token用的
AllowedGrantTypes:ResourceOwnerPassword可以使用通過用戶名密碼來換取token和ClientCredential只使用ClientSecrets來換取token
AllowedScopes:這裏只用socialnetwork
我們要對token進行簽名，所以需要一對public和pirvate key,在開發時可以設定AddDeveloperSigningCredential()
如果是發行時自已要建立一個簽名然後使用 AddSigningCredential()
